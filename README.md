# AlmaU Syllabus басқару жүйесі

Университет силлабустарын жасауға, қарауға, бекітуге және экспорттауға арналған веб-қосымша.

## Негізгі мүмкіндіктер

- Жүйе ішінде силлабустарды жасаңыз және өңдеңіз
- Дайын `PDF` және `DOCX` файлдарын жүктеңіз
- Қарап шығу алдында AI көмегімен алдын ала тексерулерді орындаңыз
- `мұғалім -> декан -> umu` бекіту ағынын қолдау
- Хабарландыруларды, өзгертулер журналын және PDF экспортын бір жерде сақтаңыз

## Рөлдер

- `мұғалім`
- `декан`
- `umu`
- `admin`

## Жергілікті жүгіру

- `start_project.bat` Windows жүйесінде жергілікті әзірлеу ортасын бастайды
- `python manage.py runserver` веб-қосымшаны іске қосады
- `python manage.py run_worker` AI/фон жұмысшысын іске қосады

## Тәуелділіктер

- `requirements.txt` негізгі орындалу уақытының тәуелділіктерін қамтиды
- `requirements-ai.txt` қашықтан/тексеру мүмкіндіктеріне арналған AI-ға қатысты тәуелділіктерді қамтиды
- `requirements-ai-local.txt` қосымша жергілікті-LLM тәуелділіктерін қамтиды
