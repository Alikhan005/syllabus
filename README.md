**AlmaU Syllabus Management System** — AlmaU университетіне арналған силлабустарды басқару платформасы.

Жүйе келесі мүмкіндіктерді ұсынады:
- силлабусты жүйе ішінде құрастыру және өңдеу;
- дайын `PDF / DOCX` файлдарын жүктеу;
- AI арқылы алдын ала тексеру;
- `teacher -> dean -> umu` келісу маршруты;
- хабарламалар, өзгерістер тарихы және PDF генерациясы.

Жүйеде келесі рөлдер бар:
- `teacher`
- `dean`
- `umu`
- `admin`

Жоба Render арқылы деплой жасауға дайын:
- репозиторийде `render.yaml` бар;
- build және start скрипттері дайын;
- GitHub-қа жаңа `push` жасалғаннан кейін Render жобаны қайта деплой ете алады.
